# Student Performance Analytics & Predictive Dashboard

## 📌 Project Overview
This project shifts the educational reporting paradigm from static tracking to **proactive academic forecasting**. Utilizing a dataset of 25,000 student records, this interactive Power BI solution analyzes how behavioral and routine factors—such as study hours, attendance, school type, and study methods—collectively impact final academic outcomes. 

Instead of relying on basic charts, this repository integrates **Regression-based Machine Learning logic** inside Power BI to forecast overall cumulative marks and subject-specific scores (Math, Science, and English).

---

## 🚀 Key Features

* **Overall Score Insights (Demo 1):** Real-time tracking and statistical mapping of study habits and digital access against overall academic benchmarks.
* **Subject Performance Matrix (Demo 2):** A granular Decomposition Tree matrix allowing educators to slice and drill down into Math, Science, and English scores based on teaching methods.
* **Early-Warning System (Demo 3):** An automated risk-mitigation table that uses conditional formatting to instantly flag at-risk students whose attendance drops below 75%.
* **AI-Powered Drivers:** Built-in *Key Influencers* visual that automatically identifies and displays the top triggers behind student success or failure.

---

## 🛠️ Tech Stack & Tools

* **Business Intelligence:** Power BI Desktop
* **Data Engineering (ETL):** Power Query (Data cleaning, type validation, and categorical transformation)
* **Analytics & Modeling:** DAX (Data Analysis Expressions) & Embedded Python Scripts (Linear Regression)
* **Data Source:** Kaggle (`Student_Performance.csv`)
